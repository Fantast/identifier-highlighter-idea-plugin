Identifier Highlighter
Identifier Highlighter is a plugin for IntelliJ IDEA 6/7 that provides dynamic highlighting of source code identifiers.  Examples of identifiers in Java would be {Class, Method, Field, Parameter, and Local Variable}.  As the mouse cursor moves over an identifier, that identifier is highlighted along with all other instances of that identifier throughout the active file.  Quick navigation between each identifier is achieved via keyboard shortcuts.  Other keyboard shortcuts are available to quickly jump to the declaration of the identifier, to return to where navigation was started, and to rename the identifier.
Keyboard Shortcuts 
Shortcut Action Definition
Shift+Alt+Down Next Identifier Move to the next highlighted identifier in the current active file
Shift+Alt+Up Previous Identifier Move to the previous highlighted identifier in the current active file
Shift+Alt+Right Start Identifier Move back to the starting identifier where first started navigating
Shift+Alt+Left Declaration Identifier Move to the identifier usage where the identifier is declared (if applicable)
Shift+Alt+R Rename Identifier Rename all occurrences of highlighted identifier (does not perform a full rename refactor such that other files that reference this identifier will not be changed)
Shift+Alt+L Lock Identifiers Lock the highlighting so that movement of the cursor will not dynamically highlight identifiers under the cursor
Shift+Alt+U Unlock Identifiers Unlock the highlighting so that movement of the cursor will dynamically highlight identifiers under the cursor
Shift+Alt+T Enable Plugin Toggle the plugin to be on or off
